144.二叉树先序遍历

发表于 2019-09-14 更新于 2019-09-15 分类于 Leetcode

题目

给定一个二叉树，返回它的 前序 遍历。

示例:

输入: [1,null,2,3]

输出: [1,2,3]
进阶: 递归算法很简单，你可以通过迭代算法完成吗？

题解

方法1 递归

方法2 非递归

非递归的方法用stack 的方式来解。

循环结束条件是stack 不为空，或者树不为空。

  • stack不为空，代表遍历过根节点
  • root不为空，代表可以继续遍历左子树.
